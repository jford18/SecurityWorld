01. Login

Ruta: /login

Descripción: Permite iniciar sesión en el Portal Administrativo con credenciales autorizadas.

Pasos:

  1. Ingresar usuario y contraseña.
  2. Presionar "Ingresar".
  3. Verificar que se muestre la pantalla de selección de consola (si aplica).
  4. Confirmar que el portal cargue correctamente.
  5. Si aparece un mensaje de acceso, contactar al administrador.

02. Dashboard

17. Cambio de Clave

Ruta: /cambiar-clave

Descripción: Permite actualizar la contraseña personal para proteger el acceso al portal.

Pasos:

  1. Ingresar la contraseña actual en el campo correspondiente.
  2. Escribir la nueva contraseña siguiendo las políticas indicadas.
  3. Confirmar la nueva contraseña repitiéndola en el campo de verificación.
  4. Guardar los cambios y esperar la confirmación del sistema.
  5. Cerrar sesión y volver a ingresar para validar que la actualización fue exitosa.
